# Google Clone

## Project Overview
This project is a **front-end replica of the Google search homepage**, created using HTML and CSS. The purpose of this project is to practice layout design, responsive web design, and UI replication of a widely recognized interface.

## Features
- **Responsive Design**: The layout adapts smoothly to different screen sizes (desktop, tablet, and mobile).
- **Navigation Bar**: Includes links for Gmail, Images, Google Apps, and profile placeholder.
- **Search Section**: Central search bar with microphone and lens icons.
- **Buttons**: "Google Search" and "I'm Feeling Lucky" buttons.
- **Footer**: Country name and informational links (About, Advertising, Privacy, Terms, etc.).
- **Hover Effects**: Interactive elements provide visual feedback on hover for better user experience.

## Technologies Used
- **HTML5**: Semantic structure and layout.
- **CSS3**: Styling, responsive design, hover effects, and grid/flex layouts.
- **Images**: Icons and logo stored locally in the `Assets/Images` folder.

## How to Run
1. Clone or download the repository.
2. Open the `index.html` file in a web browser.
3. Ensure that the `Assets/Images` folder is intact for all images to display properly.

## Learning Outcomes
- Understanding of layout replication of a real-world website.
- Practice in using flexbox, CSS grid, and responsive breakpoints.
- Implementing hover effects and interactive UI components.
- Organizing project assets and maintaining a clean project structure.
